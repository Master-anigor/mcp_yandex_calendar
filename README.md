Для работы с этим MCP-сервером необходимо создать специальный пароль приложения в Яндекс!

Перейдите на страницу https://id.yandex.ru/security/app-passwords
Нажмите "Создать пароль приложения"
Выберите "Календарь" (CalDEV) в качестве приложения и дайте ему имя
Скопируйте сгенерированный пароль и используйте его в файле .env или в конфигурации

НЕ ИСПОЛЬЗУЙТЕ основной пароль от вашего аккаунта Яндекс!
Пароль приложения — это отдельный, ограниченный по возможностям ключ доступа.
